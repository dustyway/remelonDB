# Examples

One example so far, split across two packages:

- [`todo-sync/`](todo-sync/README.md) — shared schema and a ~50-line
  sync server. **Start here**: its README has the setup, the two-window
  demo script, and a file guide.
- [`todo-sync-web/`](todo-sync-web/) — the React client, with the
  end-to-end suite CI runs.
