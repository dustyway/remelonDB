# Examples

- [`todo-sync/`](todo-sync/README.md) — an offline-first todo list,
  one schema shared by three packages: `backend/` (schema + a ~50-line
  sync server), `frontend/` (React web client with the end-to-end
  suite CI runs), and `mobile/` (React Native client, in progress).
  **Start with its README** for setup, the two-window demo script, and
  a file guide.
